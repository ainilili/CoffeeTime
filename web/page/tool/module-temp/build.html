模板构建
    系统自带一些默认字段: 
    tableName[String]：表名
className[String]：类名
classNameLower[String]：首字母小写的类名
fields[LIST]：对应表的字段
normalFields[LIST]：对应表的普通字段
primaryFields[LIST]：对应表的主键字段
{
  "fieldName": Java字段名，例如：userId
  "columnName": 数据库字段名，例如：user_id
  "methodName": 方法名，例如: UserId
  "dbType": 数据库类型
  "valueType": Java类型
  "comment": 字段注释
}

    如果以上字段满足不了您的需求，您可以在设置参数框内自定义参数
